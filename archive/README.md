# Archive

This folder is intentionally outside the active pipeline.

Contents:

- `configs/`: planning configs that are not executable by `main.py`.
- `docs/`: older design notes kept for history.
- `old_emerald_files/`: older Emerald helper scripts and derived JSON files.
- `reference/iqm_utils/`: IQM tutorial/reference notebooks.
- `reference/provided/`: original challenge/reference code.
- `unused_code/`: code versions not registered in the current pipeline.

Active code lives in `qec_pipeline/`.
Active configs live in `configs/`.
